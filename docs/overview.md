# Обзор проекта

Проект **Code Quality Assessment** предназначен для оценки качества Python-кода с использованием инструментов статического анализа и тестирования.

Основные цели:
- Проверка кода на соответствие стандартам PEP8
- Автоматический запуск unit-тестов
- Настройка CI/CD для автоматизации проверок
- Генерация отчётов о тестах и качестве кода
